# OTA server for for Amazon TV Stick 4K 1st Gen #

- Amazon Fire TV Stick 4K (mantis)

### Known bugs ###
- HDR
- Dolby E-AC-3 Audio
- Voice search (Alexa remote)
- SELinux is set to Permissive
